# RightNow — Платформа о правах человека

Современная веб-платформа для изучения и понимания прав человека во всём мире.

## Возможности

- 🗺️ **Интерактивная карта нарушений** — визуализация случаев нарушений прав человека по странам и категориям
- 📚 **Образовательный курс** — простой и понятный курс об основных правах человека с квизами
- 💬 **Анонимные истории** — безопасное пространство для обмена личным опытом

## Технологии

- React 18
- React Router DOM
- React Simple Maps
- Vite
- CSS3

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```
